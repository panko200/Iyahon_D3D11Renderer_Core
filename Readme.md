Iyahon_D3D11Renderer_Core  
製作者：Iyahon  
配布場所：https://github.com/panko200/Iyahon_D3D11Renderer_Core

## 概要

YukkuriMovieMaker4 にて動作する映像プラグインです。  
D3D11 のデプスバッファを利用して映像アイテムの前後関係を正確に描画します。  
アイテムごとに個別に有効・無効を切り替えられるため、必要な箇所だけに適用できます。  
また FXAA によるアンチエイリアス処理も内蔵しており、斜め線や文字のジャギーを軽減します。

## 使用方法

1. プラグインをインストールすると、映像アイテムのプロパティの「描画」グループに 「D3D11描画」 トグルが追加されます。
2. D3D11 のデプスソートを有効にしたいアイテムを選択し、トグルを ON にしてください。
3. 設定はプロジェクトファイルに保存され、アイテムのクローン・分割時にも引き継がれます。

## アンインストール方法

1. YMM4 を起動して `ヘルプ(H)` > `その他` > `プラグインフォルダを開く` をクリックする。
2. YMM4 を終了する。
3. `Iyahon_D3D11Renderer_Core` という名前のフォルダを削除する。

## 注意点

OS : Windows 11 (64bit)  
ゆっくりMovieMaker4 : v4.51.0.3
